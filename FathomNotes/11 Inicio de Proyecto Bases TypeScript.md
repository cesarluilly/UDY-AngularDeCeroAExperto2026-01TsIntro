# Proposito

* Esta es una introducción a TypeScript y un reforzamiento de JavaScript, donde se creará una carpeta en el escritorio llamada "Angular" para trabajar los ejercicios del curso.

Glosario:

* Bit: Es un bundler o empaquetador que permite el reemplazo de módulos en caliente, lo que significa que se pueden hacer cambios sin tener que recargar la aplicación por completo.
* TypeScript: Es el lenguaje de programación que se utilizará en este curso.

Atajos de teclado:

* Shift + Command + P: Abre la paleta de comandos de Visual Studio Code, que es muy útil y se usa constantemente.

Buenas prácticas, recomendaciones y consejos:

* Se recomienda trabajar localmente en la computadora, aunque también se puede usar StackBlitz para hacer los ejercicios sin necesidad de instalar nada.
* Cuando se cambie de directorio, es necesario ejecutar "npm install" para instalar las dependencias del proyecto.
* Para ejecutar la aplicación en modo de desarrollo, se usa el comando "npm run dev".
* Se sugiere memorizar el atajo Shift + Command + P para abrir la paleta de comandos de Visual Studio Code.
* Se puede personalizar el estilo de la aplicación modificando el archivo "styles.css", pero eso no es parte de este reforzamiento.
